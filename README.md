# Iuliia
## Install
### Maven
```xml
<dependency>
    <groupId>io.github.anominy</groupId>
    <artifactId>iuliia</artifactId>
    <version>0.3-r0</version>
</dependency>
```
### Gradle
```groovy
implementation 'io.github.anominy:iuliia:0.3-r0'
```
